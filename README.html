Как развернуть сайт на GitHub

Эта страница — локальная шпаргалка прямо внутри сайта.

1) Создай репозиторий

Например: mchs-nevsky. Загрузи в него содержимое ZIP (в корень).

2) Включи Pages

Settings → Pages → Source: Deploy from a branch → Branch: main / root.

3) Готово

GitHub даст ссылку вида https://username.github.io/mchs-nevsky.

Обновления

  • Оснащение: /data/equipment.json
  • Новости: /data/news.json
  • Тексты страниц: /*.html
  • Стили: /assets/css/styles.css
