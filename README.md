# МЧС Невский — ПСЧ №3 • ГИМС №7 (статический сайт)

## Быстрый деплой на GitHub Pages
1. Создай репозиторий (например `mchs-nevsky`)
2. Загрузи содержимое этой папки в корень репозитория
3. Settings → Pages → Deploy from a branch → Branch: `main` / folder: `/ (root)`

## Где править контент
- Новости: `data/news.json`
- Оснащение: `data/equipment.json`
- Тексты страниц: `*.html`

## Горячие клавиши
- Фокус на поиск: Ctrl/⌘ + K (на страницах с поиском)
